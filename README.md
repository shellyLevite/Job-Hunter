# JobHunter AI

This repository contains the backend foundation for **JobHunter AI**, a platform that automates job discovery, matching, and resume generation.

## Phase 1: Backend skeleton

- FastAPI backend
- Basic healthcheck endpoint
- Initial project structure

## Getting Started

```bash
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -r requirements.txt
uvicorn backend.main:app --reload
```
